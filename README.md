# REDKAM V6.3 — Mobile Cinema Cut

Versión optimizada para Chrome Android, Safari iPhone, Chrome desktop y Safari desktop.

## Publicar en GitHub Pages

Sube **todo el contenido de esta carpeta** a la raíz del repositorio `93kapo/redkam` y confirma el commit. GitHub Pages actualizará el mismo enlace:

`https://93kapo.github.io/redkam/`

Después realiza una recarga fuerte:

- Mac: `Command + Shift + R`
- Windows: `Ctrl + Shift + R`
- iPhone: cierra la pestaña y vuelve a abrir el enlace; si persiste, limpia los datos del sitio en Safari.

## Cambios principales

- Myke Towers en blanco y negro como proyecto principal.
- Campaña 01 y Campaña 04 integradas como Fashion + Commercial.
- Fuegos artificiales EMF integrados en Selected Work.
- Eliminación total de Domino's.
- Controles premium en SVG; no utiliza emojis como botones.
- Videos centrados y mostrados completos con `object-fit: contain`.
- Modal diseñado para videos horizontales y verticales en mobile.
- Safe areas para notch y barra inferior del iPhone.
- Videos H.264 / yuv420p compatibles con Safari y Chrome.
- Reproducción inline, bajo demanda y pausado automático fuera de pantalla.

## Prueba recomendada

1. Abre el enlace en Safari de iPhone.
2. Verifica el menú, constelación y scroll.
3. Abre Myke Towers, Campaña 01, Campaña 04 y Fireworks.
4. Rota el teléfono y vuelve a orientación vertical.
5. Repite en Chrome Android.
