# REDKAM V6 — Guía de contenido

## Qué material falta para convertir V6 en la edición completa

### 1. Hero flash sequence
Subir directamente aquí clips de 5–12 segundos o masters originales de:
- El mejor vuelo FPV del EMF.
- Pirotecnia / fuegos del EMF.
- Entrada o salida del escenario en FPV.
- Crowd masivo.
- Un momento humano o backstage.

Se usarán para crear un microshowreel de 5–7 segundos después del agujero negro.

### 2. Películas completas en YouTube
Subir como **No listado** y compartir el enlace:
- Myke Towers.
- Montecarlo 01.
- Montecarlo 02.
- Restaurante.
- EMF FPV.
- EMF pirotecnia.
- Fashion / modelos.

El sitio solo necesita el ID del video. Ejemplo:
`https://youtu.be/ABCDEFGHIJK` → ID: `ABCDEFGHIJK`.

### 3. Archivos directos para previews
Para cada proyecto, subir aquí un clip de 6–12 segundos. Puede pesar hasta varios cientos de MB; REDKAM lo convertirá a:
- MP4 web silencioso.
- Poster JPG/WebP.
- Recorte horizontal.
- Recorte móvil.

## Dónde cambiar un enlace de YouTube
En `index.html`, busca:

```html
data-youtube="S4hGxxwc4sY"
```

Reemplaza únicamente el ID.
