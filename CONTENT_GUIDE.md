# Guía de contenido REDKAM

## LIVE / FPV
Material de artistas, conciertos, festivales, crowd, stage, pirotecnia y vuelos de precisión.

## FASHION + COMMERCIAL
Modelos, lifestyle, producto, nightlife, hospitality y piezas de cámara ejecutadas bajo brief.

## Requisitos recomendados para previews web

- MP4 H.264
- Pixel format: yuv420p
- Audio: AAC
- Horizontal: 1280×720
- Vertical: 720×1280
- Duración ideal: 7–16 segundos
- `faststart` activado
- Poster JPG o WebP separado

Los masters completos deben permanecer en YouTube no listado o almacenamiento externo; el sitio utiliza previews ligeros.
